# syntax-to-sound

Vibe coding for music with Python + Renardo.

## What this repo is

A playground for composing and experimenting with generative music ideas in code.

## Stack

- Python 3.11+
- Renardo

## Quick start

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -U pip
pip install renardo
```

Then start building musical sketches and save them under `src/`.
